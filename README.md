## 🛠️ Tech Stack

- **Frontend:** React + Vite + Axios + TailwindCSS
- **Backend:** Node.js + Express
- **Database:** MongoDB Atlas
- **Deployment:** Render
- **CI/CD:** GitHub Webhooks → Render auto-deploy
